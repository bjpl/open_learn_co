Quick Reference Card

Essential Commands

Docker:
  Start:     docker-compose up -d
  Stop:      docker-compose down
  Rebuild:   docker-compose up -d --build
  Logs:      docker-compose logs -f [service]
  Shell:     docker-compose exec [service] sh
  Restart:   docker-compose restart [service]

Backend:
  Dev:       cd backend && uvicorn app.main:app --reload
  Test:      cd backend && pytest
  Lint:      cd backend && flake8
  Format:    cd backend && black .
  Migrate:   cd backend && alembic upgrade head
  Shell:     cd backend && python -m app.cli

Frontend:
  Dev:       cd frontend && npm run dev
  Build:     cd frontend && npm run build
  Start:     cd frontend && npm start
  Test:      cd frontend && npm test
  Lint:      cd frontend && npm run lint

Database:
  Connect:   psql -U postgres -d openlearn
  Backup:    pg_dump -U postgres openlearn > backup.sql
  Restore:   psql -U postgres openlearn < backup.sql
  Reset:     docker-compose down -v && docker-compose up -d

Health Checks:
  Backend:   curl http://localhost:8000/health
  Frontend:  curl http://localhost:3000
  Database:  pg_isready -h localhost -p 5432
